# :school: 리팩토링 2판 스터디

- 매주 화요일 오후 6시
- 미참여시 또는 스터디 미준비시 벌금 2,000원씩 걷기
- 매주 커리큘럼에 대한 강의 듣고 풀이 후 코드에 대한 설명 및 회고
